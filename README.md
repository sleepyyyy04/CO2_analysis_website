#### A full-stack website that manages and analyzes carbon intensity data

#### Created front-end with Semantic UI framework and developed back-end with Flask by using Python

#### Used RESTful to get data set from Carbon Intensity API and managed data with MongoDB database

#### Implemented parallel processing with Pyspark
